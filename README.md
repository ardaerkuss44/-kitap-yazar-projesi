📚 Kitap-Yazar Bulma Projesi
🎯 Proje Amacı
Bu proje, kullanıcıdan alınan kitap ismine karşılık yazarını bulmayı amaçlar. Kaggle’dan indirilen Türkçe Kitap Veri Seti kullanılmıştır. Proje Google Colab ortamında 4 bölüm halinde çalıştırılabilir.

🛠️ Kullanılan Teknolojiler
• 	Python 3
• 	Pandas (veri işleme)
• 	Google Colab (çalışma ortamı)
• 	Kaggle Turkish Book Data Set (dataset kaynağı)

📌 Proje Adımları
1. Dataset Yükleme

2. Veri Analizi

3. Kullanıcıdan Kitap İsmi Alıp Yazarını Bulma

4. Ekstra Özellik: Aynı Yazarın Diğer Kitaplarını Göster


📊 Örnek Çalışma
• 	Kullanıcı: “Tutunamayanlar”
• 	Sistem: “Yazar: Oğuz Atay”
• 	Ekstra: Aynı yazarın diğer kitaplarını listeler.

✅ Sonuç
Bu proje, basit bir veri işleme uygulaması olarak kitap-yazar eşleştirmesi yapar.
Sunum için uygun, anlaşılır ve 4 parçalı yapısıyla öğretici bir örnektir.
GitHub’a yüklenerek tamamlanmış bir proje olarak sunulabilir.
